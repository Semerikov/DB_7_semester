Книжный магазин
Страница логина для авторизации
страницы пользователя
	страница списка книг где в виде таблицы будут показаны задаваемые фильтрами или все книги 
	страница книги где будет представлена все информация с кратким описанием 
	страница автора информация об авторе и таблица его книг
	страница оформления заказа 
страницы администратора 
	страница пользователей таблица пользователей все можно редактировать 
	страница книг таблица книг все можно редактировать 
	

Из базы необходимо получать следующую информацию:
	Для администратора 
		добавление редактирование удаление информации о книгах.
		просмотр статистики по продажам ( Какой автор пользуется наибольшей популярностью , самая часто скачиваемая книга )
		работа с пользователями добавление редактирование удаление
	для пользователя 
		возможность поиска книги по фильтрам ( название автор цена isbn )
		оформление заказа и скачивание книги.
		редактирование своего профиля
